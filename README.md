This repository is a **100% open-source, living curriculum** designed to take you from a cloud beginner to an elite, highly paid Azure & AI Architect. We don't just teach you how to pass Microsoft certifications; we teach you how to survive "Day 2" operations, automate your infrastructure, and actually get hired.

## 🛑 The BBAG Philosophy (Why this repo exists)

1. **No "Click-Ops" Allowed:** If you aren't deploying it with Bicep, Terraform, PowerShell, or the Azure CLI, you aren't doing it right. We build with Code.
2. **"Break-Fix" Labs:** Anyone can follow a 10-step tutorial to build a Virtual Network. Real engineers get paid to fix what's broken. Our labs intentionally deploy broken environments for you to troubleshoot using Azure Monitor and KQL.
3. **1-Click Browser Labs:** No more wasting 3 hours installing Python and Terraform on your local machine. Click "Code -> Open in Codespaces" and get a free, fully configured browser IDE instantly.
4. **Portfolio Export:** Every lab ends with a specific deliverable to commit to your public GitHub or post to LinkedIn to prove to hiring managers that you can actually do the job.

> 💡 **BBAG PRO TIP:** Stop putting "Familiar with Azure" on your resume. When you finish a Sector here, you will have a public GitHub repository full of Infrastructure as Code and Architecture Decision Records (ADRs) that prove you are a practitioner, not just a test-taker.

## 🚀 How to Get Started

1. **Prerequisites:** We assume you know basic networking (IPs, Subnets, DNS) and OS fundamentals. If you don't, check out our [Prerequisites Guide](./Prerequisites.md) before starting.
2. **Fork the Repo:** Click the "Fork" button in the top right so you have your own copy to push code to.
3. **Spin up your Lab:** Open your fork, click the green `<> Code` button, and select **Open in Codespaces**. 
4. **Start Learning:** Navigate to Sector 1 and start building!
